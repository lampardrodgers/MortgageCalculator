# Loan Rate Lab - 房贷与贷款利率计算器

一个无需构建工具的纯静态网页房贷计算器，开箱即用。

## 功能特性

- **多种贷款类型** — 商业房贷 / 组合贷（商贷 + 公积金）/ 普通贷款
- **两种还款方式** — 等额本息 / 等额本金，自动对比两种方案的成本差异
- **年度汇总** — 按年聚合还款金额、本金、利息与本金占比
- **逐月明细** — 点击年份卡片展开该年每月的还款拆分
- **组合贷合并** — 商贷与公积金分别计息，统一汇总展示
- **试算模板** — 内置常见贷款场景，一键填充参数

## 快速开始

项目只包含 `index.html`、`styles.css` 和 `app.js` 三个文件，无任何依赖。

**方式一：直接打开**

双击 `index.html` 即可在浏览器中使用。

**方式二：本地服务**

```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve .
```

然后访问 `http://localhost:8000`。

## 使用说明

- 输入金额单位为 **万元**，输出金额单位为 **元**
- 利率按 **年利率（%）** 输入
- 普通贷款按固定本金逐月计息，不参与等额本息/等额本金对比
- 计算结果用于试算展示，银行实际扣款可能因计息规则、四舍五入与利率调整略有差异
